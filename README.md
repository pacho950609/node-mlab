# Instalacion

```js
npm i 
```

#Ejecucion

```js
npm run dev
```

#Rutas

| Tipo | ruta  | Requiere 'Authorization' Header
| :---: | :-: | :-: |
| Post | localhost:3000/auth/signIn | no
| Post | localhost:3000/auth/signUp | no
| Get |  localhost:3000/clima/:ciudad | si
| Get | localhost:3000/transacciones | si
